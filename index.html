üìù Professional Self-Assessment

As a dedicated computer science student, I have worked hard to acquire the skills and information required to excel in a dynamic and challenging industry. The capstone project allowed me to demonstrate my technical ability, problem-solving skills, and professionalism, all of which I have developed during my academic career. By improving my Animal Shelter Application, I exhibited my knowledge of software architecture, data structures, algorithms, and database management while also satisfying the Computer Science program's five main course goals.

My first enhancement was to improve the software design and engineering features of my Animal Shelter Dashboard. This update aligns with Course Outcome 1, since it fosters a collaborative atmosphere by providing elements that facilitate different user interactions, such as a responsive interface, intuitive navigation, and clearly defined user responsibilities. My revised user interface includes Dash Bootstrap components, which improve the user experience for both shelter volunteers and administrators. Features such as a search bar, a "Refresh Data" button, and the inclusion of "About Us" and "Contact Us" sections illustrate my ability to develop professional-quality visual communications, which meets Course Outcome 2. By incorporating full unit testing and logging functionality, I made sure that the application is user-friendly, and is also maintainable and dependable, demonstrating my ability to develop new solutions that correspond with Course Outcome 4. I also implemented data validation and error handling, displaying a security mentality compatible with Course Outcome 5, to secure the application's robustness and security.

My second enhancement aimed to increase the Animal Shelter Application's efficiency and scalability by refining data structures and algorithms. This update is consistent with Course Outcome 3, which required me to create and assess computer solutions that addressed efficiency and scalability issues utilizing algorithmic principles. I used an LRU caching method to speed up repeated searches, a hash map for breed-based lookups to reach O(1) time complexity, and binary search for sorted attributes like age and animal type to achieve O(log n) searching efficiency. I also implemented a cache size configurable feature depending on dataset size, assuring optimal performance regardless of database growth, demonstrating my ability to manage trade-offs in design decisions. These improvements were thoroughly verified using approaches such as performance benchmarking and hash map validation, proving my ability to apply well-founded methodologies and tools, as required by Course Outcome 4. Structured and professional documentation of my enhancements contributes to Course Outcome 2, demonstrating my ability to offer technically sound solutions targeted to various audiences.

My third enhancement focused on strengthening the database management capabilities of the application by integrating advanced security features such as user authentication, multi-factor authentication (MFA), and role-based access control (RBAC). These enhancements directly align with Course Outcome 5, as they demonstrate a security mindset by anticipating and mitigating potential vulnerabilities through bcrypt password hashing, QR-code-based MFA, and modular RBAC systems. I also created a login page with fields for username, password, and MFA OTP, and integrated a logout button in the layout page for session termination, I warranted a seamless and secure user experience, further supporting Course Outcome 1. My ability to deliver professional-quality solutions is evident in the user interface and structured documentation, which align with Course Outcome 2. My integration of advanced tools such as bcrypt for secure password storage and QR-code generation demonstrates my ability to implement well-founded and innovative computing practices, satisfying Course Outcome 4.

The process of enhancing this project was both challenging and rewarding. Each enhancement required careful planning, rigorous testing, and multiple iterations to ensure seamless integration into the existing system. Challenges such as synchronizing the hash map with live database operations, configuring the cache size effectively, and defining role-specific permissions were overcome through disciplined problem-solving and adherence to industry standards. The additional refinements based on professor feedback, such as the inclusion of integration tests and cache size configuration, further underscore my commitment to delivering high-quality solutions that meet professional expectations.

These enhancements made to my Animal Shelter Application represent a synthesis of the skills and knowledge I have gained throughout my academic career. These improvements demonstrate my technical proficiency, problem-solving abilities, and commitment to developing secure, scalable, and user-centered applications. By aligning with the program‚Äôs course outcomes, my capstone project reflects my readiness to contribute meaningfully in professional settings and positions me as a strong candidate in the field of computer science.

üìú My Artifact:

I chose the CS340 Client/Server Animal Shelter applicaton as my artifact because it exemplifies my ability to design and implement a functional database-driven application. This project involved creating a CRUD (Create, Read, Update, Delete) system to manage data for an animal shelter, highlighting essential skills in database management, Python programming, and user interface design. For my enhancements I will use tools such as Jupyter Notebook with Dash, PowerShell, MongoDB, Python, HTML, CSS, JavaScript, and Docker. I plan to improve its software design and engineering by implementing modular components and give the user interface a complete overhaul. In terms of data structures and algorithms, my goal is to introduce more efficient data filtering and search functionalities. I will also enhance the database by integrating advanced queries, improving data security through encryption, and introducing role-based access controls to align with industry standards. These improvements will demonstrate my growth in applying computer science principles to solve real-world problems effectively. The application will also be ran within a Docker container which will provide a consistent and isolated environment for data analysis and computational research.
