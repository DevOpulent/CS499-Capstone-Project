Throughout my academic journey in the Computer Science program, I have developed a comprehensive set of technical skills, problem-solving abilities, and a professional mindset, all of which have prepared me to address real-world challenges in the field of computer science. My capstone project serves as a culmination of these experiences, demonstrating my growth and proficiency through three key enhancements to the Animal Shelter Application. These enhancements highlight my abilities in software design, data structures, algorithms, and database management while aligning closely with the program‚Äôs course outcomes. Furthermore, I incorporated additional refinements based on my professor‚Äôs feedback to ensure the project met the highest standards of professionalism and quality.

For the first enhancement, I focused on improving the software design and engineering aspects of the Animal Shelter Dashboard. These improvements included a refined user interface, the addition of comprehensive unit and integration tests, and the integration of logging for maintainability and transparency. The updated user interface features a search bar, a refresh button, and dedicated sections for "About Us" and "Contact Us," creating an intuitive and user-friendly experience. To ensure system reliability, I developed unit tests to validate CRUD operations and edge cases and enhanced logging with performance monitoring metrics to track query execution times. These enhancements align with Outcome 1 by fostering collaborative environments that enable diverse users to interact effectively with the application. Additionally, the structured code, professional documentation, and enhanced UI demonstrate Outcome 2 by effectively communicating technical solutions. Finally, the integration of performance monitoring and testing reflects Outcome 4, showcasing the use of innovative tools and techniques to ensure reliability and maintainability.

The second enhancement concentrated on optimizing algorithms and data structures to improve performance and scalability. I implemented an LRU caching mechanism to speed up repeated queries, introduced a hash map for breed-based lookups with O(1) time complexity, and utilized binary search for sorted attributes like age and animal type, achieving O(log n) efficiency. Moreover, I incorporated a cache size configuration feature that dynamically adjusts based on dataset size, ensuring efficient memory usage regardless of the database‚Äôs scale. These enhancements align with Outcome 3, as they demonstrate my ability to design and evaluate scalable and efficient computing solutions using algorithmic principles. The detailed documentation and testing of these features align with Outcome 2, as they effectively communicate technical solutions tailored to diverse audiences. Additionally, Outcome 4 is addressed by the application of well-founded tools and techniques to implement robust and scalable solutions.

The third enhancement emphasized advanced database management and security. I developed a ‚Äúusers‚Äù collection in MongoDB to securely manage user credentials and roles, incorporating bcrypt for password hashing and implementing multi-factor authentication (MFA) using QR codes. A role-based access control (RBAC) system was established to ensure appropriate permissions for different user roles, including guests, regular users, and administrators. The enhancements also included a new login page with fields for username, password, and MFA, along with a logout button for secure session management. These updates align with Outcome 1 by creating a secure and user-friendly interface that fosters collaboration among diverse users. The professional-quality implementation and user-centered design align with Outcome 2, while the use of advanced tools like bcrypt and MFA aligns with Outcome 4 by demonstrating innovative and secure solutions. Additionally, Outcome 5 is supported by these enhancements, as they reflect a proactive security mindset through the implementation of MFA, RBAC, and secure password storage.

The process of enhancing this project was both challenging and rewarding. Each enhancement required careful planning, rigorous testing, and multiple iterations to ensure seamless integration into the existing system. Challenges such as synchronizing the hash map with live database operations, configuring the cache size effectively, and defining role-specific permissions were overcome through disciplined problem-solving and adherence to industry standards. The additional refinements based on professor feedback, such as the inclusion of integration tests and cache size configuration, further underscore my commitment to delivering high-quality solutions that meet professional expectations.

In conclusion, the enhancements made to my Animal Shelter Application represent a synthesis of the skills and knowledge I have gained throughout my academic career. These improvements demonstrate my technical proficiency, problem-solving abilities, and commitment to developing secure, scalable, and user-centered applications. By aligning with the program‚Äôs course outcomes, my capstone project reflects my readiness to contribute meaningfully in professional settings and positions me as a strong candidate in the field of computer science.

üìú My Artifact:

I chose the CS340 Client/Server Animal Shelter applicaton as my artifact because it exemplifies my ability to design and implement a functional database-driven application. This project involved creating a CRUD (Create, Read, Update, Delete) system to manage data for an animal shelter, highlighting essential skills in database management, Python programming, and user interface design. For my enhancements I will use tools such as Jupyter Notebook with Dash, PowerShell, MongoDB, Python, HTML, CSS, JavaScript, and Docker. I plan to improve its software design and engineering by implementing modular components and give the user interface a complete overhaul. In terms of data structures and algorithms, my goal is to introduce more efficient data filtering and search functionalities. I will also enhance the database by integrating advanced queries, improving data security through encryption, and introducing role-based access controls to align with industry standards. These improvements will demonstrate my growth in applying computer science principles to solve real-world problems effectively. The application will also be ran within a Docker container which will provide a consistent and isolated environment for data analysis and computational research.
